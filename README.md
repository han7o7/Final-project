# CarFixHelper

CarFixHelper is a car repair assistant website designed to help users diagnose and fix common car issues through an intuitive search interface and curated repair guides.

## Live Demo
https://han7o7.github.io/Official-Carfix-helper/

## GitHub Repository
https://github.com/han7o7/Official-Carfix-helper

## Features
-  **Search Car Issues:** Quickly search for common problems (e.g., changing oil, replacing brake pads).
- **Repair Guides:** Direct links to Wikipedia articles with step-by-step instructions.
- *Image Gallery:** Visual aids to guide users through each repair process.
-  **Contact Support:** Display a support phone number for personalized assistance (replace with your number).

## Technologies Used
- HTML5
- CSS3
- JavaScript (vanilla)
- Responsive design for mobile and desktop

## Installation
    git clone https://github.com/han7o7/Official-Carfix-helper.git
    cd Official-Carfix-helper

## Usage
1. Enter a car issue in the search bar.
2. Select a suggestion to view repair steps.
3. Refer to the images for visual guidance.
4. Call the provided support number for further help.
